###Responsive Web Desing:
* Metatag Viewport

```html
 <meta name="viewport" content="width=device-width, initial-scale=1">
```

* Medidas fluídas

```css
.content{
    display: block;
    width: 94%;
    margin: 0 auto;
    max-width: 1920px;
    min-width: 200px;
    font-size: 0,75em /*12px*/

}
```
* Media queries
* [Aspect ratio com css](http://www.mademyday.de/css-height-equals-width-with-pure-css.html)
